组播接口

 

 

接口名称

MultiCastMessage

协议

Multicast

意义

会议启动后，主持人客户端作为组播源，向路由器发送信息

组播信息

serverIP

String

服务器IP

meetingID

String

会议ID

meetingState

int

会议状态 未开始1；已开始 2；已暂停 4；已结束 8；

agendaIndex

int

议程序号

contentType

int

会议当前内容类型

documentIndex

int

文件序号

voteIndex

int

投票表决序号

voteState

int

 投票状态 未开始1；已开始 2；已结束 4；

IsUpdate

int

是否更新 未更新0；待更新 1

IsReturnVoteResult

int

是否返回表决结果 不返回0；要返回 1；

补充说明

此组播信息支持多会议室、支持掉线重连
